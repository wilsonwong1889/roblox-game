# Week 4 Blueprint

Week 4 is the expanded first-floor build target.

The runtime code in this repo is still the last validated foundation, but the docs and validation baseline now point at the next pass:

- larger first floor
- 8 first-floor droppers
- increasing costs and payouts across the first-floor lane
- starter clicker income path that matches the first dropper payout flow
- owner door control button for open/close access
- auto collector unlock after all 8 droppers
- spawn moved to a corner of the base
- `100000` test wallet for Studio verification

## Week 4 Goal

By the end of Week 4, the first floor should feel like a full factory pass instead of a small starter lane.

The visible goals are:

- more lane depth and a larger tycoon footprint
- more progression steps before the collector unlocks
- a clearer owner-only entrance
- a test wallet large enough to exercise every upgrade path

## Validation Scope

This blueprint documents the intended Week 4 build target.
Validation should continue to verify the current baseline safely while the gameplay pass catches up.
