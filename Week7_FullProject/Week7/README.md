# Week 7

Week 7 makes the tycoon persistent, competitive, and combat-polished.

## Scope

- save/load and autosave
- rebirth
- raid windows and breach flow
- collector theft cap
- start-menu shop tab
- world competition board
- 10-minute event rotation
- PvP join arena
- zombie siege rounds against claimed tycoons
- late-game defense upgrades with turret automation
- startup world validation plus emergency spawn fallback if the main map bootstrap fails
- tighter front-door shell with no walk-through gap when closed
- armory weapon pickups by holding `E`
- click-aimed guns with crosshair support
- supported sights/zoom on select guns
- expanded armory chain with shotgun and sniper tiers

## Validation Focus

- startup bootstrap: lobby, shared map, spawn, and tycoon count
- front door closure and wall-to-door seal
- armory pickup flow and duplicate pickup behavior
- click aim, crosshair visibility, and scoped-gun sights
- save/load round trip
- rebirth reset and prestige count
- raid open, breach, and theft rules
- VIP starter pack prompt from the UI
- PvP join/respawn flow during arena rounds
- zombie wall pressure during siege rounds
- alarm beacon / turret / guard tower build and firing behavior
